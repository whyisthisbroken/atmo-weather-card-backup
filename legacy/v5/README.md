# 📜 Legacy: Atmospheric Weather Card v5.x

This folder contains the last v5.x version of the card, kept for anyone who
prefers it over the current v6+ `atmo-weather-card`.

**This code is not maintained.** No fixes, updates, or support are provided
for files in this folder. For the actively maintained version, use the files
in the repository root.

For installation and configuration of this version, see the release notes
under [Releases → v5.0](https://github.com/whyisthisbroken/atmo-weather-card/releases/tag/v5.0).
