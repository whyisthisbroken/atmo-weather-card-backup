# Security Policy

## Supported Versions

This is a personal, actively maintained fork. Security fixes are only
provided for the current major version.

| Version | Supported          |
| ------- | ------------------ |
| 6.x     | :white_check_mark: |
| < 6.0   | :x:                |

## Reporting a Vulnerability

Please do **not** report security vulnerabilities through public GitHub issues.

Instead, use the private reporting channel via this repository's **Security** tab
("Report a vulnerability" / GitHub Private Vulnerability Reporting).

This is a personal project maintained on a best-effort basis — there is no
guaranteed response time, but security reports will be prioritized over
feature requests. If a vulnerability is confirmed, a fix will be released as
soon as practical; public disclosure happens only after a patch is available.
